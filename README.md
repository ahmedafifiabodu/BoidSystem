# BoidSystem
BaseBoids is an Unreal Actor that implements boid behavior: sets root, mesh and collision components; seeds a randomized velocity; computes separation via collision sweeps; calculates player attraction and orbit forces; applies steering each Tick, limits speed/force, updates location and rotation. Exposed properties tune perception, avoidance debug
